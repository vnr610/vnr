HTML Basics & Structure

Date: Dec 2025 HTML5 Day 1

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for documents designed to be displayed in a web browser. It tells the browser what to show (text, images, buttons), while CSS tells it how to show it.

The Boilerplate

Every HTML file needs a standard structure. This is the first thing I learned to type out.

<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <!-- Content goes here -->
    <p>Hello World!</p>
  </body>
</html>

Essential Tags I Learned

Most elements have an opening tag and a closing tag, but some are self-closing.

  • Text: h1-h6 (Headings), p (Paragraph), span (Inline text)
  • Buttons: <button>Click me</button>
  • Links: <a href="...">Link</a> (Anchor tag)
  • Images: <img src="..." /> (Self-closing void element)
  • Lists: <ul> (Unordered) and <ol> (Ordered) with <li> items
<!-- Button Example -->
<button class="subscribe-btn">
  SUBSCRIBE
</button>

Semantic HTML

Instead of using <div> for everything, modern HTML uses tags that describe their meaning.

  • <header> - Top of the page/section
  • <nav> - Navigation links
  • <main> - Dominant content
  • <footer> - Bottom of the page
  • <article> - Independent content (like this blog post)
