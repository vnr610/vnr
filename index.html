JavaScript Fundamentals

Working with Strings

Strings are used for storing and manipulating text. JavaScript provides many helpful methods to handle string data efficiently.

Common Methods
  • .length: Returns the length of the string.
  • .toUpperCase(): Converts string to upper case.
  • .slice(start, end): Extracts a part of a string.
  • .replace(): Replaces a specified value with another value.
Example Code 
const text = "Hello World";
                                                                    console.log(text.length); // 11
                                                                    console.log(text.toUpperCase()); // "HELLO WORLD"
                                                                    let part = text.slice(0, 5); // "Hello"

Conditional Logic

Conditional statements are used to perform different actions based on different conditions.

Types
  • if: Specifies a block of code to be executed if a condition is true.
  • else: Specifies a block of code to be executed if the same condition is false.
  • else if: Specifies a new condition to test, if the first condition is false.
  • switch: Selects one of many blocks of code to be executed.
Example Code 
let hour = 14;
                                                                    if (hour < 12) {
                                                                    console.log("Good morning");
                                                                    } else if (hour < 18) {
                                                                    console.log("Good afternoon");
                                                                    } else {
                                                                    console.log("Good evening");
                                                                    }

Loops & Iteration

Loops are handy, if you want to run the same code over and over again, each time with a different value.

For Loop

Loops through a block of code a number of times.

While Loop

Loops through a block of code while a specified condition is true.

Example Code 
const cars = ["BMW", "Volvo", "Saab", "Ford"];
                                                                    let text = "";

                                                                    for (let i = 0; i < cars.length; i++) {
                                                                    text += cars[i] + "\n";
                                                                    }

JavaScript Functions

A JavaScript function is a block of code designed to perform a particular task. A function is executed when "something" invokes it (calls it).

Arrow Functions (ES6)

Arrow functions allow for a shorter syntax for writing function expressions.

Example Code 
// Traditional
                                                                    function add(a, b) {
                                                                    return a + b;
                                                                    }

                                                                    // Arrow Function
                                                                    const multiply = (x, y) => x * y;

                                                                    console.log(multiply(5, 5)); // 25
