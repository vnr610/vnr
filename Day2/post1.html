CSS Basics & Selectors

Date: ... CSS3 Day 2

What is CSS?

CSS (Cascading Style Sheets) is what makes the web look good. Without it, everything is just plain text and ugly buttons (like the default HTML). CSS handles colors, layouts, fonts, and animations.

The Syntax

I learned that CSS rules consist of a selector and a declaration block.

p {
  color: red;
  font-size: 16px;
}

Selectors: Class vs ID

The most important thing I learned today is how to target elements.

  • Element Selector (p, button): Targets all elements of that type. Good for defaults.
  • Class Selector (.btn): Targets elements with class="btn". This is what we use 90% of the time!
  • ID Selector (#header): Targets one specific element. High specificity (hard to override).
/* Targeting a class */
.subscribe-button {
  background-color: #cc0000;
  color: white;
  border: none;
}

Colors & Backgrounds

We can use names (red), RGB values (rgb(255, 0, 0)), or Hex codes (#ff0000). Hex codes give us the specific shades we want, like the "Amazon Orange" or "YouTube Red".
